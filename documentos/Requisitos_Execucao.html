Manual de Instalação e Execução

Painel de Controle de Monitoramento (SIMRE-CEAVS)

Projeto: SIMRE-CEAVS
Arquitetura: Frontend React (Dados Mockados)
Data de Atualização: 19/11/2025
Modo Demonstração: O sistema foi configurado para rodar independentemente de Banco de Dados ou API. Todos os dados são fictícios e armazenados na memória do navegador.

Sumário

1. Pré-requisitos 2. Execução do Sistema 3. Backend e Banco de Dados (Opcional)

1. Pré-requisitos

Para executar a interface do sistema, você precisa apenas de:

  • Node.js (v18+): Necessário para rodar o ambiente de desenvolvimento React.
    Download Node.js

2. Execução do Sistema

Abra um terminal na raiz do projeto e execute:

# 1. Instalar dependências (caso seja a primeira vez)
npm install

# 2. Iniciar o Frontend
npm run dev

O sistema estará acessível em: http://localhost:5173.

Credenciais de Acesso (Mock)

  • Admin: Use qualquer email contendo a palavra "admin" (ex: admin@anvisa.gov.br) e qualquer senha.
  • Usuário Comum: Use qualquer outro email e senha.

3. Backend e Banco de Dados (Opcional)

Os arquivos do Backend (Node.js/Express) e scripts do SQL Server ainda existem na pasta backend/ e doc_banco/, mas não são necessários para rodar a versão atual da interface.

Caso deseje reativar a conexão real no futuro, será necessário reconfigurar os apontamentos de API no código do Frontend.
