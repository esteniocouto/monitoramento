2. Execução Automática (Windows)

Novo! Agora você pode iniciar todo o sistema com apenas um clique.
  1. Navegue até a pasta raiz do projeto.
  2. Localize o arquivo iniciar_sistema.bat.
  3. Dê um duplo clique no arquivo.

O que o script fará:

  • Abrirá uma janela de terminal para o Backend (instala dependências e inicia a porta 3001).
  • Abrirá uma janela de terminal para o Frontend (instala dependências e inicia a porta 5173).
  • Aguardará 8 segundos para o carregamento.
  • Abrirá automaticamente o seu navegador padrão no endereço do sistema.

3. Execução Manual

Caso prefira ou esteja em Linux/Mac, execute os comandos abaixo em dois terminais separados:

Terminal 1 (Backend)

cd backend
npm install
npm run dev

Terminal 2 (Frontend)

# Na raiz do projeto
npm install
npm run dev

Em seguida, abra o navegador em: http://localhost:5173.
